# Follow the next steps to Install, Run, Test and Deploy the API.

## Dependencies

* React
* Docker (Optional)

## Install
```
npm i
```

## Run
```
npm start
```

## Tests
### Runs all tests like in the CI environment
```
npm test
```
### Runs all tests like in the CI environment and keep watching
```
npm run test:auto
```
### Coverage tests
```
npm run coverage
```
### Lint tests
```
npm run lint
```
